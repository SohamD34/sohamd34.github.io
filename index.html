LLM Fine Tuning

Experimenting with Parameter Efficient Fine Tuning (PEFT) methods like LoRA, QLoRA to finetune Large Language Models (LLMs). The aim is to get an understanding of how LLMs function, learn pipelining of LLMs, and how models are fine-tuned to adapt to specific tasks and data

Time Series Forecating using ML

A learning-based project focused on predicting sleep patterns based on health features obtained through smartwatch data. This project aims to leverage machine learning, deep learning and time series algorithms to predict sleep quality and duration, providing insights that can be valuable for individuals seeking to improve their sleep habits.

Natural Language Processing

The aim of this project is to use make use of textual descriptions of a movie's plot/summary to classify movies into genres through BERT tokenization and classification models.

Recommendation System for E-commerce

This project aims to provide personalized recommendations for users based on their past behavior and preferences, includes analytical capabilities that allow for the exploration of user behavior and product trends, depending on factors such as location, time, etc.

Code Correction and Genration using LLMs

This is an attempt to fine tune SOTA Large Language Models like Llama, Codegen so as to generate Verilog (VHDL) programmes for prompted use case, detect syntax, logic and human errors in programmes and rectify them for generate functioning programmes.
